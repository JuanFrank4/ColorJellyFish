# ColorJellyFish
A simple practice - project I did for my Artifitial Vision class that consist in a movil robot that reacts to colors using a TCS34725 and Arduino.
The reason why calling "color jellyfish" is that I got inspired by a video I watched a long ago where explained how some jellyfish reacts and behaves with different colors.

It uses the Adafruit arduino library, here the link to the repository:
https://github.com/adafruit/Adafruit_TCS34725/
